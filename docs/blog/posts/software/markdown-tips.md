# Markdown 使用技巧记录
## 1.markdown链接如何设置新窗口打开？
### 1.1.使用特定语法
```markdown
语法：[链接文字](URL){:target="_blank"}
示例：访问 [example.com](https://example.com){:target="_blank"}
```
 

### 1.2.使用 HTML 标签
```markdown
语法：<a href="URL" target="_blank">链接文字</a>
示例：访问 <a href="https://example.com" target="_blank">example.com</a> 
```
